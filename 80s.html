Windows 1.0 (1985)
Inicialmente, Windows era poco más que un complemento del entorno operativo para DOS. Esta versión de windows contaba con herramientas básicas como lo son Calculadora, Calendario, Fichero de tarjetas, Visor del portapapeles, Reloj, Panel de control, Bloc de notas, Paint, Reversi, Terminal y el predecesor de Word, Write y a pesar de esto el sistema no gozó de gran popularidad.

Windows 2.0 (1987)
Windows 2.0 abrió a los programas la memoria expandida que normalmente se guardaba para ejecutar periféricos. Supuso una mayor capacidad multitarea y empezó a parecerse más a la imagen real con Microsoft colocando ventanas superpuestas en el diseño. El sistema introdujo nuevos atajos de teclado y las ya famosas aplicaciones Excel y Word, lo cual hizo que estuviera al mismo nivel de Apple.
