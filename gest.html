FUNCIONES
Y
GESTIONES

Como se mencionó anteriormente, Windows es el sistema operativo desarrollado por Microsoft, el cual tiene como función principal ser un mediador entre el usuario y la máquina, facilitando la conexión entre ambos. Actualmente, el más reciente es el "Windows 11" y este está basado en Windows 10, lo más reciente que ha traído este Windows es que nos proporciona una interfaz de usuario nueva y una experiencia de usuario optimizada.

A continuacion se mostraran las funciones de Windows ...



FUNCION 1

Manejo de los recursos de la computadora


Es la función primaria del sistema operativo de Windows ya que maneja y organiza los recursos de la computadora los cuales llegan a ser el CPU, RAM y el disco duro.El Administrador de recursos del sistema de Windows solo administra los recursos del procesador cuando la carga combinada del procesador es superior al 70 por ciento esto significa que no limita activamente los recursos que se pueden utilizar.

How to manage the resources
FUNCION 2

Provee una interfaz



Windows ofrece una GUI que nos permite acceder y utilizar varias aplicaciones y programas de una manera sencilla ya que esto permite a los usuarios seleccionar menús, hacer clic en iconos y navegar fácilmente por varias aplicaciones.

GUI Windows Information
FUNCION 3

Gestion de Archivos



Windows nos facilita la gestión de archivos así como el data storage, la configuración de derechos de acceso y la búsqueda de archivos ya que Windows proporciona varias herramientas para ayudar a los usuarios a organizar y almacenar datos y archivos en sus computadoras o portátiles todo esto gracias a el El Shell de Windows 2000 permite asociar archivos con un usuario en particular, de modo que los archivos permanezcan ocultos para otros usuarios.

Tricks for Managing Your Files
FUNCION 4

Facilitar la configuración de la Red



Este sistema provee varias herramientas para facilitar la conexión como la network configuration, firewall configuration y el network access configuration lo que hace posible que el usuario pueda conectarse a redes y compartir información con otros dispositivos que están en la misma red de manera más fácil ya que Windows tiene APIs, componentes, y servicios que dan mantenimiento a nuestras apps usando las redes y el internet.

Windows Network
FUNCION 5

Facilita la Seguridad


Este sistema operativo provee varias funciones de seguridad así como los antivirus, antimalwares y firewalls que ayudan a proteger nuestro dispositivo de ataques de los virus y malwares debido a que este siempre a sido su objetivo, pero el windows actual nos ayuda a proteger nuestra identidad al usar métodos avanzados de seguridad para iniciar sesión en tu dispositivo.

Windows Security Information
FUNCION 6

Proporciona Compatibilidad


Este sistema está diseñado para aceptar varios depósitos de hardware y softwares, haciendo que de manera más sencilla los usuarios puedan instalar y utilizar diferentes aplicaciones y programas en el dispositivo pero con el paso de los tiempos aun se espera que la compatibilidad con aplicaciones de escritorio (Win32) sea segura, con la mayoría de las aplicaciones existentes trabajando sin cambios.
